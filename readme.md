# 키지의 개인 과외

## Motivation
중고급 Front-end Developer로의 성장을 위해 핵심적인 Publishing 능력과, Javascript, 웹 생태계를 보다 깊게 파악하는것이 중요 핵심. 또, 학원생들과 보다 차별 적인 학습과, 코드에 대한 고찰과 반복학습으로 Code Review를 통한 실력 향상에 입각한 개인 과외입니다.
